Carpeta para activos de la web (imágenes, logos, etc.).

Estructura recomendada:
- `logos/` → versiones del logo en PNG/SVG (ej: `logo.svg`, `logo-dark.svg`)
- `img/` → imágenes de contenido y banners

Instrucciones:
- Añade aquí los archivos gráficos y referéncialos desde HTML con rutas relativas, por ejemplo: `../assets/logos/logo.svg` desde `docs/idiomas/es/`.
